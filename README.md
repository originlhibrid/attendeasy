# Smart Attendance Tracker

A simple, web-based attendance tracking tool that helps students keep track of their class attendance and calculate the required number of classes to meet their attendance goals.


## Features

- Add Subjects: Easily add subjects and set your target attendance percentage.
- Track Attendance: Mark attendance for each class as "Present" or "Absent."
- Calculate Requirements: View how many more classes you need to attend or can afford to skip to reach your target percentage.
- Persistent Storage: Attendance data is saved in the browser's local storage, so your data remains even after refreshing the page.
